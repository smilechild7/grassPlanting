[명령어 정리]

ls : 현재 디렉토리의 파일과 디렉토리 목록 표시

pwd : 현재 작업 중인 디렉토리 출력

cd 디렉토리_이름 : 다른 디렉토리로 이동할 때 사용

mkdir 디렉토리_이름 : 새 디렉토리 생성

rmdir 디렉토리_이름 : 비어있는 디렉토리 삭제

touch 파일_이름 : 빈 파일을 생성

rm 파일_이름 : 파일이나 디렉토리를 삭제 (주의하여 사용할 것)

cp 원본 대상 : 파일이나 디렉토리를 복사

mv 원본 대상 : 파일이나 디렉토리를 이동하거나 이름을 변경함

cat 파일_이름 : 파일 내용을 연결하여 표시

more 파일 이름 
less 파일 이름 : 파일 내용을 한 번에 한 화면씩 표시하여 스크롤 할 수 있게 함
